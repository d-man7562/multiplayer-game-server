INSTRUCTIONS FOR MY GAME

MOVE: W A S D for quad-directional movement 
SHOOT: SPACEBAR + W A S D to shoot in one of four directions. 
when pressing SPACE, you cannot move until you have 'shot' in a direction using WASD.
If you hit a shot, your score will increase by 1.
If you get hit, you will respawn in a random spot on the map.
CHAT: Enter + message + enter to send mesesage
PRIATE CHAT: Enter + recipient name + message + enter
TAB: show entire scoreboard
scoreboard automatically updates for an individual if they score.

To connect to a online server, if you are not LAN to the server, you must manually configure the IP via the HOST variable   (line 13 client.py). Hopefully this doesn't cause trouble, because it saved me a lot of time while debugging. I just confirmed today that it works using multiple computers to connect to the same server. 

Do not worry about setting up map, everything is automated. If you want to add your own tile map, make sure it is formed in lines similar to what I have now. Also, make sure the ROWS and COLS variable matches the rows and colummns in your new map.



I hope you enjoy this game. My mom and I had a blast playing this game (online!) for the first time today just now.
